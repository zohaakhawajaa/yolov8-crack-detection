# YOLOv8 Crack Detection

This repository contains an implementation of **crack detection in images using YOLOv8**. It includes training and evaluation scripts, performance metrics such as **precision, recall, confusion matrix**, and visualizations of model performance.  

The project is ideal for learning object detection workflows and evaluating detection models on crack datasets.  

---

## Features

- Object detection using **YOLOv8**  
- Evaluation metrics:  
  - Precision & Recall  
  - Confusion Matrix  
  - Plots for metrics visualization  
- Ready-to-use scripts for training and evaluation
